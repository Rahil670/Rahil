RippleX Firebase-Ready Setup

1. Firebase Hosting:
   - Run `firebase login`
   - Run `firebase init` (choose Hosting and select existing project: ripplex-c114a)
   - Run `firebase deploy`

2. Firebase Auth Enabled:
   - Google
   - Email/Password
   - Phone (OTP)

3. Files:
   - index.html: Main App
   - firebase.js: Firebase config
   - firebase.json: Hosting config

You're ready to deploy!
